# Text Generation with Markov Chains

## Overview
This project demonstrates text generation using Markov Chains in Python. The model is trained on a sample dataset and generates new sentences based on learned word patterns.

## Objective
To understand and implement probabilistic text generation using the Markovify library in Google Colab.

## Technologies Used
- Python
- Google Colab
- Markovify

## Project Workflow
1. Install the Markovify library.
2. Create a training dataset.
3. Build a Markov Chain model.
4. Generate new text sentences.
5. Save the generated output to a text file.

## Files Included
- `Text_Generation_Markov_Chains.ipynb` - Google Colab Notebook
- `generated_text.txt` - Generated text output
- `README.md` - Project documentation

## Sample Output

```
Artificial Intelligence is transforming industries across the globe.
Machine Learning enables computers to learn patterns from data.
Technology continues to evolve at a rapid pace.
```
